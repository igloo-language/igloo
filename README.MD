## An Example

```igloo
func return_five() {
  5 // or `return 5;`
}
x = return_five();
// `x` now has the value of `5`
```

## Getting Started

To use Igloo, download this repository locally. E.g:

```bash
cd igloo # go into igloo repository folder
python3.8 igloo/ igloo_file.ig # filename
```
